Bienvenido a mi página de ejemplo

Acerca de esta página

Esta es una página HTML para repasar conceptos y hacer pruebas.

Características

  • Con <h1> a <h6> definimos encabezados de distintos tamaños.
  • Con <p> introducimos texto en párrafos.
  • Con <strong> ponemos el texto en negrita.
  • Con <em> ponemos el texto en cursiva.
  • Con <span> creamos un identificador útil para estilos.
  • Con <br> realizamos un salto de línea.
  • Con <hr> insertamos una línea horizontal.
  • Con <a href="URL"> creamos enlaces.
  • Con <img src="imagen.jpg" alt="Texto alternativo"> insertamos imágenes.
  • Con <table>, <tr>, <td> creamos tablas.
  • Con <li> definimos elementos de una lista.
  • Con <ol> y <ul> creamos listas ordenadas(numeros) y desordenadas(puntos).
  • Con <form> creamos formularios para entrada de datos.
  • Con <input> insertamos campos de entrada en formularios.
  • Con <button> creamos botones interactivos.
  • Con <div> agrupamos elementos para estructurar la página.
  • Con <nav> definimos una barra de navegación.
  • Con <article> representamos contenido independiente.
  • Con <aside> definimos contenido lateral, como widgets o barras laterales.
  • Con <blockquote> citamos texto de otra fuente con formato especial.

Imágenes en html

Se usa src para establecer donde esta la imagen

Se usa alt para un texto por si no carga

Se usa title para asignarle un titulo a la foto
